Pill-shaped call-to-action button — use for the main action in any view ("Get Started Today", "Book a Dispatcher").

```jsx
<Button variant="primary" size="lg" iconRight={<ArrowIcon />}>Get Started Today</Button>
```

Variants: `primary` (electric-blue gradient, default), `energy` (logo crimson — reserve for the single highest-intent CTA), `secondary` (outline on dark), `ghost` (text only). Sizes: `sm` / `md` / `lg`. Always fully rounded (pill). Hover brightens; press nudges down + scales slightly.
