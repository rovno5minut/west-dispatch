Compact uppercase status pill — use for load states, feature tags, and eyebrow labels.

```jsx
<Badge tone="success" dot>Loaded</Badge>
<Badge tone="accent">24/7 Dispatch</Badge>
```

Tones: `neutral`, `accent` (blue), `energy` (crimson), `success` (green / loaded), `warning` (amber / in-transit). Set `dot` for a glowing status dot. Mono font, wide tracking.
